Parts of the Box Model

  • Content: The actual text or elements inside the box.
  • Padding: Space between the content and border.
  • Border: The visible outline of the element.
  • Margin: Space outside the border separating elements.

Try toggling between box-sizing: content-box and box-sizing: border-box in the CSS to see the difference.
