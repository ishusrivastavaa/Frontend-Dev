Responsive Design Principles

Fluid Layouts

Fluid layouts use relative units like percentages instead of fixed pixels, allowing elements to resize with the viewport. 

container {
  width: 90%;
}

Flexible Media

Images and videos scale smoothly within their containers using max-width and height settings. 

img {
  max-width: 100%;
  height: auto;
}

Breakpoints

Breakpoints define specific screen widths where layout or styling changes for better usability. 

@media (max-width: 600px) {
  body { font-size: 14px; }
}
